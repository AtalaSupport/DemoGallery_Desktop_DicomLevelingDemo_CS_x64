# DicomLevelingDemo
This is a very basic demo that shows how to open and view a DICOM image using our 
DicomImage and DicomDataset classes to enable Window and Level control in a Winforms app.  

If you're looking for display controls on DICOM images such as zoom, magnify and panning, 
or if you're looking for examples of viewing metadata on DICOM images, check out our Dicom 
Viewer Demo.  

Requires evaluation or purchased licenses of DotImage Document Imaging or Photo Pro, as 
well as our Dicom Decoder addon.

This is the C# version


## Prerequisites
This demo assumes you have the Atalasoft DotImage SDK installed and licensed for 
DotImage Document Imaging. Optionally, you may license DotImage Photo Pro
with the DICOM add-on, but that is a legacy option for customers who did not 
purchase DotImage Document Imaging.

You may also request a 30 day evaluation when installing / activating.

[Download DotImage](https://www.atalasoft.com/BeginDownload/DotImageDownloadPage)
